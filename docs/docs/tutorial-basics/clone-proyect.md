# Desarrollo del Proyecto

Este apartado describe los primeros pasos para trabajar con el proyecto de **gestión de documentos académicos**.

## Clonación del Proyecto

Para empezar a trabajar, necesitas clonar el repositorio desde GitHub. Abre **PowerShell** (o tu terminal preferida) y ejecuta el siguiente comando:

```bash
git clone https://github.com/juan2344/ultimoparcial.git
```
